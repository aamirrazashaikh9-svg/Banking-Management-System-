# Banking System

## Project Description
This project is a Java-based Banking Management System developed for the Object-Oriented Programming (OOP) semester project. The system manages customers, accounts, transactions, and reports using Java Swing and MySQL database connectivity.

## Features
- User Login System
- Customer Management
- Account Management
- Deposit and Withdraw System
- Transaction Management
- Reports Generation
- Database Connectivity
- GUI Interface using Java Swing

## OOP Concepts Used
- Classes and Objects

- Java
- Java Swing
- JDBC
- MySQL

- model package
- dao package
- gui package
- util package
- interfaces package
- events package

## How to Run
1. Open the project in IntelliJ IDEA or NetBeans.
2. Import the SQL file into MySQL database.
3. Configure database settings in DBConnection.java if needed.
4. Run Main.java to start the application.
